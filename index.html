Progressive Web Apps is a web application that utilizes a variety of modern web features so that it can present a user experience like a native application. PWA changes the display content that is generally opened through a browser page into a separate application window. In addition PWA also allows page content to be accessed in offline mode, displaying notification messages, to access to hardware from the device as well as the native app. Progressive Web App can present web applications that are reliable, fast and maintain a bond with the user.The Class Building Progressive Web Apps was developed by Codepolitan in collaboration with Dicoding.
